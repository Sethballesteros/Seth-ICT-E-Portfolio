Lesson 4 - CSS Rules and Classes

CSS Rule

A CSS rule has two parts: a selector and one or more style declarations.

Inline example: <tag style="property:value;">

Internal/External example: selector {property:value;}

Style Properties

  • Fonts
  • Colors and backgrounds
  • Text
  • Boxes and layout
  • Lists
  • Tag classifications

Classes

A class is a user-defined selector to apply styles to specific HTML tags. It is reusable and flexible.

This text is styled with the "highlight" class.

This text uses two classes: highlight + note.

Element + Class Selector

Example: p.note {color:blue; font-size:18px;}

This paragraph is styled only if it is a <p> with the class "note".

Grouping Selectors

You can group selectors with commas. Example:

h1, p {color: purple;}
