Lesson 1-Introduction to CSS

What is CSS?

CSS stands for Cascading Style Sheets. It describes how HTML elements are displayed on screen, paper, or other media. CSS separates content from design, making websites easier to maintain and more attractive.

Why Use CSS?

  • Defines styles for webpages including layout and design.
  • Provides consistency across pages.
  • Saves time: one CSS file can control multiple pages.
  • Improves accessibility and responsiveness.
  • Enhances design with layout, colors, and animations.

Layers of a Webpage

  1. Content Layer: Defines text, images, video, and other content using HTML tags.
  2. Presentation Layer: Styles the webpage using CSS for colors, layout, and readability.
  3. Behavior Layer: Adds interactivity with JavaScript.

Advantages of CSS

  • Saves time and effort.
  • Faster loading and downloading.
  • Better design flexibility.
