Lesson 5 - CSS Selectors

What are CSS Selectors?

CSS selectors are used to find or select the HTML elements that you want to style.

They are divided into six types:

  1. Class Selector
  2. Element (Tag) Selector
  3. ID Selector
  4. Universal Selector
  5. Group Selector
  6. Attribute Selector

Element Selector

It selects all HTML elements of a specific type (like <p>, <div>, or <h1>) and applies styles to them.

p {line-height: 1.5; font-size: 16px;}

ID Selector

Used to style a single unique element on the page. Defined using the # symbol and the element's id attribute.

Example: #header {background-color: navy; color: white;}

HTML: <div id="header">Welcome to my website</div>

Universal Selector

The universal selector (*) applies styles to all elements on the page.

* {margin: 0; padding: 0; box-sizing: border-box;}

Group Selector

Groups multiple elements so you can apply the same styles at once.

h1, h2, h3 {color: darkgreen; text-align: center;}

Attribute Selector

Styles elements based on their HTML attributes and values.

input[type="text"] {border: 1px solid gray;}
input[type="submit"] {background-color: green; color: white;}

When to Use Each Selector

  • Class Selector: For styling multiple elements with the same design.
  • ID Selector: For styling one unique element.
  • Element Selector: For styling all of a specific tag type.
  • Universal Selector: For global styles across all elements.
  • Group Selector: To apply the same style to several different tags.
  • Attribute Selector: To style elements based on their attributes.

Tip: Combine selectors smartly to make your CSS organized and powerful!
