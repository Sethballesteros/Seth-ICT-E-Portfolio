Lesson 3 - Inline, Internal, External CSS and Display

Inline Style Sheets

Inline CSS is applied directly inside an HTML element using the style attribute.

(Inline CSS Example: This text is styled inline.)

Internal (Embedded) Style Sheets

Internal CSS is written inside a <style> tag in the <head> section of an HTML document. It applies styles to the entire page.

External Style Sheets

External CSS is stored in a separate .css file. It allows consistent styling across multiple pages by linking the stylesheet using <link> in the head.

Display Property

The display property controls how elements are displayed. Common values include:

  • block - elements displayed vertically (e.g., <p>, <div>)
  • inline - elements displayed horizontally (e.g., <span>, <a>)
  • list-item - displays like a list element (<li>)
  • none - hides the element completely

Block vs Inline Elements

Block elements take up full width and start on a new line (e.g., <div>, <p>, <h1>).
Inline elements only take up as much width as needed and stay in the same line (e.g., <span>, <a>, <strong>).

Div and Span Tags

<div> is a block-level container used to group large sections of content.

<span> is an inline container used for small chunks of text or inline styling.
