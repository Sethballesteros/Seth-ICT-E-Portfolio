Structure of CSS

Selector: The HTML element you want to style.

Property: The style attribute you want to modify.

Value: The setting or configuration assigned to a property.

Declaration: A single instruction that defines a style.

Declaration Block: A block of code that contains one or more declarations.

The Style Sheet and Its Parts

A style sheet is a set of instructions to a web browser on how to display elements.

(Inline CSS Example: This sentence uses orange text.)

Three Kinds of CSS

  1. External Style Sheets: Stored in a .css file and can be reused on multiple pages.
  2. Embedded Style Sheets: Defined inside <style> tags in the head of an HTML file.
  3. Inline Style Sheets: Defined directly in the style attribute of an HTML tag.

CSS Comments

Comments start with /* and end with */. They are ignored by browsers.
