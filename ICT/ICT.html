Lesson 1-Introduction to CSS

CSS (Cascading Style Sheets) describes how HTML elements are displayed. It separates content from design, allowing consistency and easier updates.

Lesson 2-Style Sheets

There are three ways to add CSS:

  • External Style Sheet: separate .css file
  • Internal Style Sheet: inside <style> tags
  • Inline Style: inside HTML elements

Lesson 3-Display Property

The display property defines how elements appear (block, inline, none, etc.). <div> is block-level, <span> is inline.

Lesson 4-CSS Rules and Classes

A CSS rule has a selector and declarations. Example: p { color: blue; }

Classes are reusable styles for multiple elements.

Lesson 5-CSS Selectors

  • Class Selector: .classname
  • ID Selector: #idname
  • Element Selector: p, h1
  • Attribute Selector: input[type="text"]

Lesson 6-Dimensions & Pseudo-classes

Use width and height to set sizes. Pseudo-classes like :hover and :active change styles based on user interaction.
